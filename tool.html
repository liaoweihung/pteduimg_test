新增圖卡產生器
這個工具不會直接上傳到 GitHub。它會幫你產生「要放到 cards.json 的片段」和建議檔名。你仍需要把圖片上傳到 img/，再把 JSON 貼進 cards.json。

    4. 實際上架步驟

    1. 到 GitHub 的 img/ 資料夾，上傳上方列出的圖片檔案。
    2. 打開 cards.json，把產生的 JSON 片段貼進最外層大括號內。
    3. 注意前一筆資料後面要有逗號。
    4. Commit changes。
    5. 打開網站，用 Ctrl + Shift + R 強制刷新。
