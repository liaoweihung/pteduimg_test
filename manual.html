📘 圖卡後台維護說明

🎯 基本概念

你只需要會用 cards.json 就可以管理全部圖卡。

其他檔案：

  • build.py → 不用碰
  • cards.manual.json → 不用碰
  • js / sw.js → 不用碰
---

👁️ 隱藏圖卡

在 cards.json 找到圖卡：

"Neupro": {
  "title": "Neupro 貼片",
  "hidden": true
}

✔ hidden: true → 不顯示

❌ hidden: false 或沒寫 → 會顯示

---

📊 排序圖卡

"Neupro": {
  "order": 40
}

數字越小 → 越前面

  • 10 → 很前面
  • 100 → 很後面
---

➕ 新增圖卡

直接複製一個：

"new_card": {
  "title": "新圖卡",
  "category": "pharmacist_product",
  "order": 50,
  "icon": "💊"
}
---

📂 分類說明

  • pharmacist_general → 通用藥品
  • pharmacist_product → 特定產品
  • public_education → 民眾衛教
---

⚠️ 常見錯誤

改了沒變？

  • 👉 按 Ctrl + Shift + R
  • 👉 用無痕模式
  • 👉 等 30 秒（GitHub Pages 延遲）
---

🚑 緊急還原

如果壞掉：

GitHub → commits → 回到上一版
---

💡 最重要原則

  • 只改 cards.json
  • 不要動其他檔案
  • 出問題先刷新
